# StepAside brand asset

`AppIcon-1024.png` is the project master. The iconset and `.icns` are regenerated by `scripts/generate-icons.sh`.

The final master was created with the built-in OpenAI image-generation tool, then locally converted from a flat chroma-key surround to transparent PNG and inspected at 1024, 128, 32, and 16 pixels.

## Final generation prompt

> Create an original, realistic three-dimensional macOS app icon for StepAside, a utility that pushes many open windows into a precise non-overlapping arrangement. Place one premium warm-ivory rounded-square icon tile on a flat chroma-key background. On the tile, four dark graphite window slabs resolve outward from a compact, slightly offset center stack into a precise 2-by-2 edge-aligned arrangement. Keep the center transition extremely simple so the silhouette stays clear at 16 px. Use restrained yellow, pale-blue, coral, and ivory window faces; realistic anodized graphite metal, matte enamel, crisp geometry, subtle bevels, and ambient occlusion only within the tile. Use a centered straight-on composition with slight three-quarter depth. No text, numbers, arrows, hands, people, watermark, copied marks, background scene, or extra objects.
